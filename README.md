E5R Deployer
============

The simple deployer to replace your hand.

## Prepare development environment:

Unix:
```sh
$ ./venv.sh
$ . ./VENV/bin/activate

(VENV) $ pip install -r requirements.txt 
```

Windows CMD:
```bat
C:\> venv.cmd
C:\> .\VENV\Scripts\activate.bat

(VENV) C:\> pip install -r requirements.txt
``` 

## Generate dist
Unix:
```sh
$ ./build.sh
```

Windows CMD:
```bat
C:\> build.cmd
```
